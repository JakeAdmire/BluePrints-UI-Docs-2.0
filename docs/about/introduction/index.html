#What is blueprints-ui?

A cleaner, better design system for ReactJS

A small UI component library and a way to keep me active and up-to-date on the best practices for front-end software development & design.

#Who made it?

Jake Admire, a front-end software developer from Colorado.

I'm very passionate about two things: development & design. So I was obviously excited about the thought of building a component UI library.
